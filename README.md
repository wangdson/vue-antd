# antd-demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### 技术点介绍
本项目使用到vue + vue-router + vuex数据流
ui框架 vue-ant-design
建议使用utils工具类 详细如下
data.js 数组或者对象的 增删改
dedounce.js 防抖操作
promiseWrapper.js promise 封装
index.js 各种工具类
